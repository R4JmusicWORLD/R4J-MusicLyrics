# YouTube Cookies Setup Guide

## কেন YouTube Cookies দরকার?

Cloud servers (Railway, Heroku, Render) থেকে YouTube direct access করলে "Sign in to confirm you're not a bot" error আসে। Cookies দিয়ে এই সমস্যা solved হয়।

---

## পদ্ধতি ১: COOKIES_TXT Environment Variable (সবচেয়ে সহজ - Railway/Heroku এর জন্য)

### Step 1: YouTube Cookies Export করুন

1. **Chrome Extension Install করুন:**
   - [Get cookies.txt LOCALLY](https://chrome.google.com/webstore/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc)
   
2. **YouTube.com এ Login করুন** (যেকোনো Google account দিয়ে)

3. **Extension ক্লিক করুন** → `youtube.com` select করুন → **Export** ক্লিক করুন

4. `youtube.com_cookies.txt` file download হবে

### Step 2: Cookies File Content Copy করুন

File খুলুন এবং **সম্পূর্ণ content copy করুন**। এরকম দেখতে হবে:

```
# Netscape HTTP Cookie File
# https://curl.se/docs/http-cookies.html
# This file was generated by libcurl! Edit at your own risk.

.youtube.com	TRUE	/	TRUE	1762934400	VISITOR_INFO1_LIVE	abc123xyz
.youtube.com	TRUE	/	TRUE	1762934400	PREF	f1=50000000
...
```

### Step 3: Railway/Heroku তে Environment Variable Set করুন

#### Railway:
1. Railway Dashboard → Your Project → **Variables** tab
2. **New Variable** ক্লিক করুন
3. Key: `COOKIES_TXT`
4. Value: **পুরো cookies file এর content paste করুন** (multi-line supported)
5. **Save** করুন
6. Service automatically restart হবে

#### Heroku:
1. Heroku Dashboard → Your App → **Settings** tab
2. **Config Vars** → **Reveal Config Vars**
3. KEY: `COOKIES_TXT`
4. VALUE: Cookies content paste করুন
5. **Add** ক্লিক করুন

**Note:** যদি multi-line paste issue হয়, তাহলে সব newlines (`\n`) দিয়ে replace করে single line বানান:
```bash
# Mac/Linux:
cat youtube.com_cookies.txt | tr '\n' '\\n'
```

## পদ্ধতি ২: Direct File Upload (VPS/Docker এর জন্য)

### Step 1: Repository তে cookies/ folder তৈরি করুন

```bash
cd MusicLyrics
mkdir -p cookies
```

### Step 2: Cookies File Copy করুন

```bash
cp ~/Downloads/youtube.com_cookies.txt cookies/youtube.txt
```

### Step 3: Commit করুন (Optional - public repo হলে **করবেন না**)

**Warning:** Public repository তে cookies commit করবেন না! Private repo বা VPS এ direct upload করুন।

```bash
git add cookies/youtube.txt
git commit -m "Add YouTube cookies"
git push
```

## Cookies Valid কিনা Test করুন

Bot চালান এবং `/play` command দিয়ে কোনো গান play করার চেষ্টা করুন। Logs check করুন:

```bash
# Railway/Heroku:
# Dashboard এ "Logs" tab check করুন

# VPS:
journalctl -u musiclyrics -f
```

**Success logs:**
```
✅ Loaded 1 cookie file(s)
✅ Innertube WEB+cookies: stream obtained
```

**Failure logs:**
```
⚠️ No cookie files found. YouTube may block requests
```

## Cookie Expire হলে কী করবেন?

Cookies সাধারণত **6-12 months** valid থাকে। Expire হলে:

1. YouTube থেকে **re-login** করুন
2. নতুন cookies export করুন
3. `COOKIES_TXT` environment variable **update** করুন
4. Service restart করুন

## Troubleshooting

### ❌ Still getting "No video formats found" error

**Solution 1: YouTube Proxy যোগ করুন**

Cookies alone সবসময় কাজ নাও করতে পারে cloud servers এ। Proxy + Cookies = Best solution.

```bash
# Railway/Heroku Environment Variables:
YOUTUBE_PROXY=ip:port:username:password
```

**Proxy providers:**
- [Webshare.io](https://www.webshare.io/) - $2.99/month (10 proxies)
- [IPRoyal](https://iproyal.com/) - Residential proxies
- [Bright Data](https://brightdata.com/) - Premium but reliable

**Solution 2: VPS ব্যবহার করুন**

VPS (Oracle Cloud, DigitalOcean) থেকে YouTube সাধারণত কোনো issue ছাড়াই কাজ করে:

- [Oracle Cloud Free Tier](https://www.oracle.com/cloud/free/) - Forever free, 1-4 CPU/24GB RAM
- [DigitalOcean](https://m.do.co/c/abc123) - $6/month

## Additional Tips

1. **Multiple Cookies:** একাধিক cookies files যোগ করতে পারেন (rotation হবে):
   ```
   cookies/youtube1.txt
   cookies/youtube2.txt
   cookies/youtube3.txt
   ```

2. **Cookie Freshness:** নতুন cookies বেশি reliable। প্রতি 3 months এ refresh করুন।

3. **Different Google Accounts:** বিভিন্ন Google accounts এর cookies use করতে পারেন (rotation)

4. **Test URL:** Browser এ YouTube থেকে logout করে test করতে পারেন cookies কাজ করছে কিনা
